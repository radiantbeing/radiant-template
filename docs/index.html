RADIANT TEMPLATE

What is RADIANT TEMPLATE?

RADIANT TEMPLATE is a defined template of the basic theme of my project.

How to use

Git clone the radiant-template repository to your project folder. The radiant_template folder must be located in the top-level directory of the project. Please refer to docs/index.html to configure the website after finishing git clone.

References

It is based on normalize.css and Skeleton by default.

Font

We adopted Pretendard as the default font. Pretendard is the best font that is neat, clear and easy to see. Dynamic subsets are applied to free up webfont capacity pressures.
