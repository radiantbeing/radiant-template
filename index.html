RADIANT TEMPLATE

RADIANT TEMPLATE란?

RADIANT TEMPLATE은 개인 프로젝트인 RADIANT 시리즈에 공통적으로 적용되는 웹 라이브러리입니다. 미리 정립된 CSS 스타일과 자동 생성되는 네비게이션 바로 쉽게 웹 사이트를 생성해보세요.

시작하기
  1. Github에서 RADIANT TEMPLATE의 레포지토리의 소스를 다운로드하세요.
  2. 소스 내의 index.html을 가이드로 프로젝트를 진행하세요.
